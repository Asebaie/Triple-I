# Рекомендательная система для Netflix Prize

## Описание

Этот проект представляет собой рекомендательную систему, разработанную на основе **автоэнкодера** с использованием PyTorch. Цель системы — предсказывать рейтинги, которые пользователь поставил бы фильму, и на основе этих предсказаний формировать персонализированные рекомендации.

## Функциональность

*   **Обучение модели:** Загрузка и предобработка датасета Netflix Prize, обучение автоэнкодера с оптимизацией гиперпараметров с помощью Optuna.
*   **Генерация рекомендаций:** Выдача топ-K рекомендованных фильмов для конкретных пользователей на основе истории просмотров.

## Структура проекта

*   `train.py`: Обучение автоэнкодера с использованием Optuna.
*   `recommend.py`: Генерация рекомендаций для пользователей на основе обученной модели.
*   `artifacts/`: Папка для сохранения обученной модели и отчёта.
*   `logs/`: Папка для файлов логов.

## Установка и запуск

### 1. Подготовка данных

1.  Скачайте датасет Netflix Prize с [Kaggle](https://www.kaggle.com/datasets/netflix-inc/netflix-prize-data).
2.  Распакуйте архив и поместите `archive` в корень проекта.

### 2. Обучение модели

1.  Убедитесь, что установлены все зависимости.
2.  Запустите `train.py`.

*Процесс обучения может занять значительное время. Результатом будет обученная модель, сохранённая в `artifacts/`, и файл отчёта.*

### 3. Генерация рекомендаций

1.  Убедитесь, что обученная модель (`ae_model_bundle_optuna.pkl`) находится в папке `artifacts/`.
2.  Запустите `recommend.py`: Скрипт выведет по одной рекомендации для 10 пользователей.

### 4. Бонус

1. Для предварительного просмотра `ae_model_bundle_optuna.pkl` запустите файл `read_pkl.py`

---

*Если у вас возникли проблемы с кодировкой файлов датасета, убедитесь, что они сохранены в кодировке `latin-1`.*
